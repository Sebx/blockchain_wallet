## Installation

1. Make sure [Python 3.6+](https://www.python.org/downloads/) is installed. 

2. Install requirements.  

```
$ pip install -r requirements.txt
``` 

5. Run the server:
    * `$ export FLASK_APP=app.py`
    * `$ export FLASK_DEBUG=1`
    * `$ flask run ` 
    * `$ flask run -p 5001`
    * `$ flask run --port 5002`
